# 🔐 Web Security Scanner

An automated Web Security Scanner developed to analyze website vulnerabilities and generate a security score using advanced security checks.

## 🚀 Features

- 🌍 **Domain & DNS Validation**
- 🔌 **Port Scanning (80, 443, 21, 23, 3389)**
- 🔒 **SSL/TLS Certificate Analysis**
- 🔎 **WHOIS Lookup**
- 🛡️ **Security Headers Analysis**
- ⚡ **Server Information Detection**
- 🔄 **HTTP → HTTPS Redirect Check**
- 📊 **Security Score & Risk Level (Low / Medium / High)**

---

## 🧠 Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Libraries Used:**
  - `requests`
  - `socket`
  - `ssl`
  - `certifi`
  - `python-whois`

## ⚙️ Installation & Setup

**1. Clone Repository**
```bash
git clone https://github.com/Priyanshukrsingh/web-security-scanner.git
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Run Application**
```bash
python app.py
```

**4. Open Browser**
[http://127.0.0.1:5000](http://127.0.0.1:5000)
